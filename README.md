# logcheck

- Линтер для Go, который проверяет лог-сообщения на соответствие правилам стиля и безопастности.
- Совместим с golangci-lint.

- Шаг 3 (тестирование) был сделан по больше части с использованием LLM Claude opus, так как с написанием тестов LLM справляются очень хорошо, мной была проверена корректность тестов и внесены доработки.

- Также с помощью LLM был создан отдельный проект для проверки работоспособности сделанного линтера. Код проекта и результат проверки представлены в директории `LLM-test-project`
## Правила

1. **lowercase** - лог-сообщения должны начинаться со строчной буквы (есть авто-исправление)
2. **english** - лог-сообщения должны быть на английском языке
3. **special-chars** - лог-сообщения не должны содержать спецсимволы и эмодзи
4. **sensitive** - лог-сообщения не должны содержать чувствительные данные

## Поддерживаемые логгеры

- `log/slog` (стандартная библиотека)
- `go.uber.org/zap` (Logger и SugaredLogger)
